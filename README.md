## Keybindings
* `q` => Quit

* `h` => Move left
* `l` => Move right
* `j` => Move down
* `k` => Move up

* `=` => Zoom in
* `-` => Zoom out

You can also use quantifiers
* `10j` => Move 10 steps up
* `6=` => Zoom in 6x

* `space` => Reset to default position and zoom
